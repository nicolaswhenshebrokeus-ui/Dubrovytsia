# Рекомендована структура папки images

Створіть таку структуру папок для ваших зображень:

```
images/
│
├── favicon.png                    # Іконка сайту (32x32px або 64x64px)
│
├── hero-background.jpg            # Головне фото на першій сторінці
│                                  # Рекомендований розмір: 1920x1080px або більше
│                                  # Формат: JPG (оптимізоване для веб)
│
├── card-history.jpg               # Фото для картки "Історія"
├── card-photos.jpg                # Фото для картки "Фотоархів"  
├── card-records.jpg               # Фото для картки "Метричні книги"
│                                  # Рекомендований розмір: 800x600px
│
└── gallery/                       # Папка для фотоархіву
    ├── photo-1.jpg                # Історичне фото 1
    ├── photo-2.jpg                # Історичне фото 2
    ├── photo-3.jpg                # Історичне фото 3
    └── ...                        # Додайте стільки фото, скільки потрібно
                                   # Рекомендований розмір: 1200x900px
```

## Вимоги до зображень

### Фонове зображення (hero-background.jpg)
- **Мінімальний розмір:** 1920x1080px
- **Рекомендований розмір:** 2560x1440px або більше
- **Формат:** JPG
- **Характеристики:** Атмосферне, історичне фото
- **Приклади:** Старе фото міста, архівний документ, історична будівля

### Картки на головній сторінці
- **Розмір:** 800x600px (співвідношення 4:3)
- **Формат:** JPG
- **Вага файлу:** До 300KB (оптимізуйте для веб)

### Фотографії в галереї
- **Розмір:** 1200x900px (або оригінальний розмір історичних фото)
- **Формат:** JPG
- **Важливо:** Зберігайте оригінальну якість історичних фото

## Де знайти зображення

### Для тестування сайту:
1. **Unsplash.com** — безкоштовні високоякісні фото
2. **Pexels.com** — безкоштовні стокові зображення
3. **Pixabay.com** — безкоштовні фото та ілюстрації

### Для реального сайту:
1. Власні історичні архіви
2. Державний архів Рівненської області
3. Приватні колекції мешканців
4. Місцеві музеї та бібліотеки
5. Сімейні фотоархіви

## Оптимізація зображень

Перед завантаженням на сайт, оптимізуйте фото:

### Онлайн інструменти:
- **TinyPNG.com** — стиснення без втрати якості
- **Squoosh.app** — від Google, детальні налаштування
- **ImageOptim** — для Mac

### Рекомендації:
- Стискайте JPG до якості 80-85%
- Використовуйте сучасні формати (WebP) для кращого стиснення
- Для історичних фото зберігайте оригінали окремо

## Як додати зображення на сайт

1. Покладіть файл в правильну папку (див. структуру вище)
2. Запам'ятайте точну назву файлу
3. У HTML файлі знайдіть відповідний `<img>` тег
4. Змініть `src="images/назва-файлу.jpg"`

**Приклад:**
```html
<img src="images/gallery/dubrovytsia-1920.jpg" alt="Дубровиця, 1920 рік">
```

## Важливо!

- ✅ Називайте файли англійськими літерами (photo-1.jpg, не фото-1.jpg)
- ✅ Використовуйте малі літери
- ✅ Замість пробілів використовуйте дефіс (hero-background.jpg, не hero background.jpg)
- ✅ Додавайте описовий alt текст для кожного зображення
- ✅ Зберігайте оригінали історичних фото в окремій папці

## Резервне копіювання

Завжди зберігайте копії оригінальних історичних фотографій:
- На окремому жорсткому диску
- У хмарному сховищі (Google Drive, Dropbox)
- На DVD/Blu-ray дисках для довготривалого зберігання
